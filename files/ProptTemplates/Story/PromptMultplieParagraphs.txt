I am import paragraph into my database via a JSON file. They must be propberly formatted to be imported.

- inside the value of an attribute quote characters must be escaped with a backslash
- inside the content of a paragraph, line breaks must be replaced with the string "\n"


This is the format a paragraph must match for import:

{
  "paragraphs": [
    {
      "htmlcontent": "",
      "sortnumber": null,
      "content":"",
      "name":"",
      "applicationincluded": null,
      "applicationexcluded": null
    }
  ]
}
